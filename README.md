## Installation

```bash
wget https://raw.githubusercontent.com/mapi68/flac-trim/master/flac-trim.conf
wget https://raw.githubusercontent.com/mapi68/flac-trim/master/flac-trim
chmod +x flac-trim*
./flac-trim
```
